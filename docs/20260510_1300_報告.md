# Godotプロジェクト新規作成 ～ GitHub Desktop連携 手順

# 目的

以下を考慮した開発環境を構築する。

- Godot 4
- 2Dアクションゲーム
- GitHub運用
- GitHub Pages公開
- WebExport対応
- 厳格なブランチ管理
- 学習用プロジェクト

---

# 1. プロジェクト名決定

今回は名前を統一する。

| 項目 | 名前 |
|---|---|
| Project Name | CoinCollector2D |
| GitHub Repository | coin-collector-2d |
| ローカルフォルダ | coin-collector-2d |

---

# 2. Godotプロジェクト作成

## Godot起動

`New Project` を選択。

---

## 入力内容

| 項目 | 設定 |
|---|---|
| Project Name | CoinCollector2D |
| Project Path | 任意の開発フォルダ |
| Renderer | Compatibility |

---

## Renderer設定理由

Compatibility を使用することで：

- Web公開安定性向上
- 軽量化
- ブラウザ互換性向上
- 学習時の描画トラブル減少

を狙う。

---

# 3. プロジェクト初期設定

## Window設定

`Project > Project Settings > Display > Window`

### 設定

| 項目 | 値 |
|---|---|
| Width | 1152 |
| Height | 648 |
| Stretch Mode | canvas_items |
| Stretch Aspect | keep |

# 5. GitHub Repository作成

| 項目 | 内容 |
|---|---|

| License | 任意 |

# 7. Git初期化

Git未初期化の場合。

```text

```

を選択。

## Repository設定

|---|---|
| Name | coin-collector-2d |
| Local Path | Godotプロジェクト |
| Git Ignore | None |
| License | None |

# 8. .gitignore 作成

Godot用 `.gitignore`

```gitignore
.godot/
.import/
export.cfg
export_presets.cfg

*.translation

# Mono
.mono/
data_*/

# System
.DS_Store
Thumbs.db
```

保存先：

```text
coin-collector-2d/.gitignore
```
